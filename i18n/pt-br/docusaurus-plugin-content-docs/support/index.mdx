## Alguma Pergunta?
Se você tiver perguntas, perceber que algo está faltando ou tiver ideias para melhorar o
livro prático, por favor, [registre um problema](https://github.com/faithtechcreate/workbook) para nós.

## Envolva-se
Este é um projeto de código aberto, então há muitas maneiras de se envolver. Aqui estão
algumas ideias para você:

- Percorra o livro prático e nos informe se algo está faltando [registrando um problema](https://github.com/faithtechcreate/workbook) 
ou conversando com nossa comunidade no Slack.
- Olhe os [problemas em aberto](https://github.com/FaithTechCreate/workbook/issues). 
Dê soluções e sugestões que possam resolvê-los.
- Se encontrar algum problema que você gostaria de corrigir, faça um pull request. Problemas
identificados como [bom problema inicial](https://github.com/FaithTechCreate/workbook/issues?q=is%3Aissue+is%3Aopen+label%3A%22good+first+issue%22) 
são um bom lugar para começar.
- Leia o livro prático. Se encontrar algo que está confuso ou que pode ser melhorado, você
pode clicar em "Editar essa página" no final da maioria dos documentos, o que te levará à
interface do GitHub para fazer e propor mudanças.
- Dê uma olhada nos recursos solicitados por outros na comunidade e considere abrir um pull
request se perceber que é algo em que você gostaria de trabalhar.

Nós aceitamos receber ideias e contribuições que aprimoram o livro prático. Verifique o
[projeto no GitHub](https://github.com/faithtechcreate/workbook) para mais detalhes.

## Junte-se à Comunidade
Junte-se ao canal `#create-general` no [Slack](https://faithtechhub.slack.com/) para fazer
perguntas e obter ajuda de outros na comunidade executando um Create.
