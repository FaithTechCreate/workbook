# Exercícios

Você examinará os exercícios para o estágio Descobrir.

## Exercício 1: Identificação do Problema

1. Identifique um problema em que você deseja trabalhar. Verifique a 
[página de projetos](/prepare/projects) para melhor orientação.
2. Escreva uma breve descrição do problema (2-3 frases). Comece com um discurso 
de desafio que enuncie o problema para seus parceiros, caso tenha.
3. Liste pelo menos três grupos de pessoas que são mais afetados por este problema.

## Exercício 2: Prática do Quebrantamento

Siga estes passos para formar um quebrantamento para o problema que você identificou:

1. Dirija-se a Deus
2. Descreva o problema a ele
3. Relembre seus testemunhos da fidelidade de Deus
4. Rogue a ele
5. Peça a intervenção de Deus
6. Agradeça e glorifique pelo que ele irá fazer

Trabalhe o quebrantamento como uma equipe. Sugerimos que você converse sobre isso e tenha
alguém para anotar.

  <summary>(Opcional) Copie e personalize o formulário Google</summary>

Faça isso se quiser customizar e salvar os resultados em seu Google Drive.

1. **Copie o Formulário:**
   - Abra o [modelo do Formulário Google](https://docs.google.com/forms/d/1RzNEWu0LICqH6CsLB7dHvXAMINLQJpCgtJNZhj4yT5k/copy).
   - Clique em "Fazer uma cópia."

2. **Personalize o Roteiro:**
   - Abra o Formulário Google copiado.
   - Acesse o menu kebab (o ícone de três pontos verticais no canto direito superior) e
   clique em `Apps Script`.
   - Substitua `YOUR_FOLDER_ID` na função `createPDF` pelo seu próprio ID de diretório do 
   Google Drive.

3. **Execute a Função `createTrigger`:**
   - Selecione a função `createTrigger` do menu dropdown e clique no botão de executar 
   (ícone de "play").

4. **Autorize o Roteiro:**
   - Você verá uma caixa de diálogo que dirá "Authorização obrigatória." 
   Clique em "Revisar permissões."
   - Você verá um aviso que diz "O Google não verificou este app."
   - Clique em "Avançado."
   - Clique em "Acessar [nome de seu projeto] (não seguro)."
   - Marque as opções e clique em "Continuar" para fornecer as permissões necessárias.

5. **Teste o Formulário:**
   - Envie uma resposta para o formulário para garantir que o PDF foi gerado e salvo em
   seu diretório específico do Google Drive.
